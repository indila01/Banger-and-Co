Banger and co

Banger and Co is a car rental application develope using MERN stack with advance technologies.

Technologies 

Backend (NodeJs)
	•	Express
	•	BcryptJS
	•	JWT
	•	MongoDB
	•	Multer
	•	Node mailer service

Frontend (ReactJs)
	•	Material Bootstrap 
	•	React Bootstrap
	•	Paypal SDK
	•	React router dom
	•	React scripts
	•	Redux
	•	Redux thunk
	•	Cypress testing
